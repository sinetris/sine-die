## Welcome to Sine Die

This user guide shows you how to get started creating technical documentation
sites using the Sine Die theme, including [basic setup](get-started), customizing
the [look and feel](look-and-feel) of your site, adding [content](content), and
making the most of Sine Die templates.

### What is Sine Die?

Sine Die is a theme designed for technical documentation using the [Hugo](https://gohugo.io)
static site generator.

The generated pages follow semantic HTML best practices and aim to be as accessibility-friendly
as possible.
